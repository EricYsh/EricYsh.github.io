前言

注：我的系统是 macOS 10.15.17（macOS 11 更新后命令可能不适用，但我都放了官网的链接，应该能正常进行。如果不行我更新到macOS11后再来修改这篇文章）

2.2 安装 iTerm2

我们要用 iTerm2 代替 macOS 自带的 终端。

  • 去 iTerm2 的官方下载：https://www.iterm2.com/index.html 需要拖拽至 Application安装

  • 或者直接用Homebrew 命令安装：brew cask install iterm2

3 安装 zsh

打开命令行输入： brew install zsh

7 配置 iTerm2 字体和配色

路径：iTerm2>Preferences>Profiles>Text>Change Font

先在上面那个红色箭头的 Non-ASCII Font 打钩，然后把下面的字体改成 Meslo （挑喜欢的那个一种就行）

路径 iTerm2>Preferences>Profiles>Colors

选择自己喜欢的配色即可（鼠标移上去 iTerm2 就会实时更改配色，选自己喜欢的即可）

8 配置 agnoster 主题

  • oh-my-zsh已经内置该主题,不用再单独下载了，直接配置就行了

命令行输入：vi ~/.zshrc

找到 ZSH_THEME 改成 angnoster

修改完成后按 ESC 键，并输入 :wq 回车。

关闭 iTerm2 再打开后即可看到最新效果。
