如何用hexo写博客

发表于 2020-04-20 更新于 2020-04-21 阅读次数： Valine：
本文字数： 636 阅读时长 ≈ 1 分钟

Create a new post

1. command line

hexo new "title"

生成新文章在 \source_posts\title.md
用markdown编辑器打开可以编辑

2. Direct Creation

也可以不使用命令自己创建 .md 文件，只需在文件开头手动加入如下格式 Front-matter 即可，写完后运行 hexo g 和 hexo d 发布。

Insert a picture

在博客根目录的 source 文件夹下新建一个 img 文件夹专门存放图片, 在博文中引用的图片路径为 /img/图片名.后缀
